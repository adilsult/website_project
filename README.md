# Adil Project Website - Modern Clothing Store

A minimalist, high-performance e-commerce frontend layout

## Structure
- **HTML5**: Semantic structure using BEM naming conventions.
- **CSS3**: Variable-based styling, Flexbox, and Grid layouts.
- **JS**: Vanilla JavaScript for DOM manipulation.

## Getting Started
1. Clone the repo.
2. Open `index.html` in your browser.

## Design Principles
- Clean Typography (Inter / San Francisco)
- Glassmorphism Navigation
- Interactive Product Cards
